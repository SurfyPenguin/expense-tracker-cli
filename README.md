# Expense-Tracker (CLI)
Expense-Tracker (CLI) is a simple expense tracker application built with java. This project is made to learn basics of java such as OOP & File I/O for persistence.

---
## How to use repository
- Clone repository
```bash
git clone https://github.com/SurfyPenguin/expense-tracker-cli/
```
- Compile the source (Use IDE instead)
```bash
javac src/*.java 
```
- Run the code
```bash
java -cp src Main
```
